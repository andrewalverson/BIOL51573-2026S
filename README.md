## BIOL51573 (spring 2026)

The class will introduce students to the basics of scientific computing, including:
	
	- projects and file organization
	- Unix
	- remote computing
	- version control with `git` and GitHub
	- programming in Python
	- workflow automation

The [schedule of topics](https://docs.google.com/spreadsheets/d/1pwmSKUIcT7qfrJ1BgCzj3-ugr64ScWsBco2Ra4QLSTs/edit?usp=sharing) is approximate and subject to change based on class progress and needs.
